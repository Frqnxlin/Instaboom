# Instaboom
Instagram follower bot

# Dependencies
[requests](https://pypi.python.org/pypi/requests)

# Usage
`python instaboom.py <username without @/post id> <number of threads> <proxy file name> <mode l/f>`

# Modes
* l = like
* f = follow
